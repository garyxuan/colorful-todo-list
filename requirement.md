# 云同步功能
1. 云同步按钮：
   - 未登录状态：
     - 弹出邮箱输入框
     - 新用户自动创建账号
     - 显示同步进度动画
   - 已登录状态：
     - 显示当前邮箱账号
     - 显示上次同步时间
     - 提供退出登录选项

2. 自动同步：
   - 已登录用户编辑后自动触发同步
   - 同步失败自动重试（最多3次）
   - 可配置关闭自动同步

3. 安全机制：
   - 邮箱验证
   - 简单密码保护
   - 数据加密传输

4. 性能优化：
   - 同步节流（最小间隔5秒）
   - 批量同步处理
   - 离线缓存支持
